[![Actions Status][actions-badge]][actions-link]

A great package that analyses penguin populations.

## Installation

```bash
python -m pip install penguins
```

From source:
```bash
git clone https://github.com/sophie145/Penguins
cd Penguins
python -m pip install .
```
